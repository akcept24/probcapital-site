# Probcapital Design System

## Brand
- **Name**: Probcapital
- **Tagline**: Trade With Our Capital. Keep Your Profits.
- **Vibe**: Ultra-premium, VIP, dark luxury fintech

## Colors
- Background: #0F1117 (near black)
- Surface: #1A1D27 (card bg)
- Surface2: #22263A (slightly lighter)
- Border: rgba(255,255,255,0.07)
- Teal Primary: #00D4AA
- Teal Light: #00FFCC
- Gold Gradient: linear-gradient(135deg, #C9A84C, #E8C96A, #C9A84C)
- Text Primary: #F5F5F5
- Text Secondary: #8A8A9A
- Text Muted: #4A4A5A
- Accent Green: #22C55E (profit/success)
- Accent Red: #EF4444 (loss indicator)

## Typography
- Font: Inter (system), fallback sans-serif
- Hero: 80px / bold / tight tracking
- H2: 48px / semibold
- H3: 28px / semibold
- Body: 16px / regular / 1.6 line-height
- Caption: 13px / regular / muted

## Spacing
- Section padding: 120px vertical
- Container max-width: 1200px
- Card padding: 32px
- Border radius: 12px (cards), 8px (buttons), 4px (badges)

## Effects
- Gold border glow: box-shadow 0 0 20px rgba(201,168,76,0.2)
- Card hover: subtle gold border + slight lift
- Glassmorphism: backdrop-blur + rgba bg on nav
- Gradient text: gold gradient on key headlines

## Components
- CTA Button: gold gradient bg, black text, 16px bold
- Secondary Button: transparent + gold border
- Stat cards: dark surface, gold number, subtle border
- Feature cards: icon + title + desc, hover gold accent
- Testimonial cards: avatar + quote + rating stars
